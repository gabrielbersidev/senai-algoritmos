Materiais {
camiseta,
estampa,
prensa térmica ou ferro
}

2-Verificar Materiais
Se faltar algum material:
  providenciar
Senão:
  prosseguir para o passo 3

3-Preparar Camiseta
Posicionar a camiseta corretamente

Colocar a estampa no local desejado

5-Aplicar Calor
Aplicar calor pelo tempo recomendado

6-Verificar Estampa
Se estampa fixada:
  prosseguir
Senão:
  reaplicar calor
